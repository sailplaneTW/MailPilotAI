# MailPilot AI

MailPilot AI is a Chrome Extension designed to enhance Gmail composition workflows using AI.

It integrates directly into Gmail's compose window and provides tools for rewriting, translating, and validating emails before sending.

---

## Core Value Proposition

Most AI email tools are standalone.

MailPilot is different:

- Works **inside Gmail**
- No copy-paste workflow
- Adds **real-time writing assistance + safety guard**

---

## Features

### 1. Email Optimization
Rewrite email content into a more professional, concise, and structured format.

### 2. Translation
Translate email body into a target language with one click.

### 3. Subject Generation
Generate a clear and relevant subject line based on email content.

### 4. Email Review (Custom Prompt)
Run a custom analysis prompt to validate tone, clarity, or risk.

### 5. Send Guard (Critical UX Feature)
Prevent accidental sending by requiring a second click within 5 seconds.
