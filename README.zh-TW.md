<h1 align="center">MailPilot AI</h1>

**MailPilot AI** 是一款頂級的 Chrome 擴充功能，利用 Google Gemini AI 的強大功能徹底改變您的 Gmail 體驗。它不僅僅是另一個 AI 工具；它是一個無縫整合的寫作助手，直接鑲嵌在您的 Gmail 撰寫視窗中，幫助您寫得更好、更快且更安全。

---

## ✨ 核心功能

### 🚀 智能寫作套件
*   **郵件優化 (Email Optimization)**：立即將草稿重寫為專業、精練且結構清晰的商務通訊。
*   **主旨產生 (Subject Generation)**：根據您的郵件內容，讓 AI 為您構思完美的主旨行。
*   **一鍵翻譯 (One-Click Translation)**：無需離開 Gmail，即可將草稿無縫翻譯成世界主要語言（英文、中文、日文等）。
*   **自定義郵件審閱 (Custom Email Review)**：使用您自己的 AI 提示詞 (Prompts)，在點擊寄出前檢查語氣、清晰度或潛在風險。

### 🛡️ 安全與生產力
*   **寄信防呆 (Send Guard)**：雙重點擊確認機制，防止意外寄出錯誤郵件。開啟後，您有 5 秒的時間可以再次確認決定。
*   **智能自動隱藏**：AI 面板會智能偵測撰寫狀態，在您寫信時出現，完成後自動隱藏，保持介面簡潔。
*   **備份與還原**：每一次的 AI 優化都會自動建立臨時備份。對結果不滿意？只需一鍵即可還原回原始文字。

### 🌐 全球化與現代設計
*   **多語系支援**：完整的 UI 語系翻譯，支援 **英文**、**繁體中文** 以及 **簡體中文**。
*   **頂級 UI/UX**：採用「毛玻璃 (Glassmorphism)」美學設計的現代化、可拖動且可調整大小的介面，完美契合您的專業工作流。

---

## 🛠️ 技術棧 (Technical Stack)

*   **Manifest V3**：基於最新的 Chrome 擴充功能標準構建，兼顧安全性與效能。
*   **Google Gemini API**：由 `gemini-1.5-flash`（或您選擇的模型）提供動力，實現頂尖的自然語言處理。
*   **原生 JS & CSS (Vanilla JS & CSS)**：輕量且快速，無外部套件依賴，確保最大程度的相容性。

## 🚀 入門指南

### 1. 安裝步驟
1.  複製此儲存庫 (Clone) 或下載原始碼。
2.  開啟 Chrome 並導覽至 `chrome://extensions/`。
3.  開啟右上角的 **開發者模式 (Developer mode)**。
4.  點擊 **載入未打包 (Load unpacked)** 並選擇此專案資料夾。

### 2. 配置設定
1.  點擊瀏覽器工具列中的 MailPilot 圖示，或前往擴充功能的「設定 (Options)」頁面。
2.  輸入您的 **Gemini API Key**（可在 [Google AI Studio](https://aistudio.google.com/) 免費取得）。
3.  點擊 **更新模型 (Update Models)** 以獲取並選擇您偏好的 AI 模型。
4.  自定義您的 **優化提示詞 (Optimize Prompts)**，或留空以使用高品質的預設值。
